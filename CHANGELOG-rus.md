# Журнал изменений

Все значимые изменения проекта фиксируются в этом файле.

Формат основан на Keep a Changelog и адаптирован для данного репозитория.

## [1.0.0] - 2026-03-04

### Добавлено

- Первый публичный релиз `Consul Viewer TUI`
- Read-only terminal UI для Consul на базе `urwid`
- `Dashboard` со сводкой по кластеру и локальному агенту
- Раздел `Telemetry` на основе метрик агента в формате Prometheus
- Разделы `Services` и просмотр инстансов сервисов
- Разделы `Nodes` и просмотр инстансов на нодах
- KV browser с preview и полным viewer
- Список сессий и просмотр деталей
- ACL read-only разделы для токенов, политик, ролей и auth methods
- Текстовая фильтрация, status filter, структурный фильтр инстансов и сортировка для каждого вида списка
- Фоновая загрузка, TTL-кэш, stale-state и popup-диагностика
- Пользовательские инструкции, README-файлы и PlantUML-диаграммы архитектуры
